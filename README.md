# PokeScan Landing - Astro

## Ejecutar en local
```bash
npm install
npm run dev
```

## Compilar
```bash
npm run build
npm run preview
```

## Desplegar en GitHub Pages
1. Crea un repo llamado `pokescan-landing`.
2. Sube este proyecto.
3. En `astro.config.mjs`, cambia `TU_USUARIO` por tu usuario de GitHub.
4. En GitHub: Settings > Pages > Source: GitHub Actions.
5. Haz push a `main`.
